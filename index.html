Welcome to Nginx with Docker!

What is Nginx?

Nginx is a high-performance web server, reverse proxy, and load balancer. It's known for its ability to handle a large number of concurrent connections efficiently.

How is Nginx Deployed Using Docker on Windows?

Deploying Nginx with Docker on a Windows machine is easy and powerful. With Docker Desktop, you can run Nginx in a container, isolating it from your system, and deploy it in seconds.

Steps to Deploy Nginx on Windows Using Docker:

  • Install Docker Desktop: First, download and install Docker Desktop for Windows from Docker's official site.
  • Run Docker Desktop: Open Docker Desktop and make sure it is running properly on your system.
  • Pull the Nginx Docker Image: Use the command docker pull nginx to download the latest Nginx image.
  • Run the Nginx Container: Start the container with the command docker run -d -p 8080:80 --name nginx-server nginx.
  • Access Your Nginx Server: Open a browser and navigate to http://localhost:8080 to see Nginx running!

How to Update the `index.html` File from Your Local Machine:

If you want to replace the `index.html` file with a new one from your local machine, follow these steps:

  • Prepare Your New `index.html` File: Ensure the new `index.html` file is ready on your local machine. For example, it's located at C:\path\to\your\new\index.html.
  • Copy the File into the Container: Run the following Docker command to copy your new `index.html` file into the Nginx container: 
    docker cp C:\path\to\your\new\index.html nginx-server:/usr/share/nginx/html/index.html
  • Verify the Update: After copying the file, open a browser and go to http://localhost:8080 to see the updated content. You can also verify it inside the container by running: 
    docker exec -it nginx-server cat /usr/share/nginx/html/index.html
